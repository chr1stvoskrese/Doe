# Screenshots

Drop the images for the README gallery here, using these exact filenames:

| Filename | What to capture | Tips |
|---|---|---|
| `board.png` | Main Kanban board with several columns and real cards | Dark theme, priorities visible, realistic data (not "test / asdf") |
| `card.png`  | An open task card — Markdown description with live preview, a checklist, a deadline | Show the editor's depth |
| `graph.png` | The task relationship graph (D3.js force-directed) | The most visually striking shot |
| `stats.png` | Weekly statistics / calendar with time tracking | Demonstrates a finished product |

**Optional extras** (add and wire into the gallery if you like): `board-light.png`
(same board in light theme), `search.png` (global search with boolean queries),
`drag.gif` (a short 5–8 s drag-and-drop clip).

## Capture tips

- Use a Retina / 2× display so images stay crisp.
- Keep a consistent window size and, ideally, the native macOS window frame.
- PNG for stills, GIF (or MP4) for motion.

## Publishing the gallery

Once the four files are in place, open `README.md` (and `README_RU.md`), find
the `## Screenshots` section, and remove the `<!-- ... -->` markers around the
gallery `<table>` to make it visible.
