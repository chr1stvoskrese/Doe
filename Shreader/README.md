# Shreader

Настольное приложение для проверки **орфографии и пунктуации русского языка** в заметках.
Вставляете текст (можно Markdown) — получаете исправленную версию с правильными кавычками-«ёлочками».

Проверка двухуровневая:

1. **Алгоритмическая** — мгновенно, офлайн:
   - орфография по словарю русского языка (pyspellchecker) с подсказками исправлений;
   - типографские кавычки: `"слово"` → `«слово»`, вложенные — `„лапки“`.
2. **LLM (Gemma 4 через Ollama)** — пунктуация и контекстные ошибки,
   которые словарь не видит. Всё выполняется локально на вашем маке,
   текст никуда не отправляется.

Markdown при этом не ломается: код-блоки, `инлайн-код`, URL и HTML-теги
не проверяются и не изменяются.

## Установка (macOS)

1. Python 3.10+ (с python.org или `brew install python-tk` для Tkinter).
2. Зависимости:

   ```bash
   pip install -r requirements.txt
   ```

3. [Ollama](https://ollama.com) и модель Gemma 4:

   ```bash
   brew install ollama
   ollama pull gemma4        # или gemma4:e2b для слабого железа
   ```

   Без Ollama приложение тоже работает — доступна алгоритмическая проверка.

## Запуск

```bash
python3 -m shreader
```

- Вставьте текст в левое поле.
- **«Проверить (алгоритм)»** — словарная орфография + кавычки. Опечатки
  подчёркиваются слева и перечисляются с подсказками внизу.
- **«Проверить (алгоритм + LLM)»** — то же самое, затем текст уходит в Gemma,
  которая правит пунктуацию. Изменения подсвечены в правом поле.
- **«Скопировать результат»** — исправленный текст в буфер обмена.

Имя модели можно поменять в поле «Модель Ollama» (например, `gemma4:26b`).

## Тесты

```bash
python3 -m unittest discover -s tests
```

## Структура

| Модуль | Назначение |
|---|---|
| `shreader/app.py` | Окно приложения (Tkinter) |
| `shreader/speller.py` | Словарная проверка орфографии |
| `shreader/quotes.py` | Расстановка «ёлочек» и „лапок“ |
| `shreader/markdown_guard.py` | Защита Markdown-фрагментов от правок |
| `shreader/llm.py` | Клиент Ollama (Gemma 4) для пунктуации |
